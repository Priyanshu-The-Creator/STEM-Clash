#🎮 STEM Clash

STEM Clash is a web-based Tic Tac Toe game created as a project for the School STEM Club. The game features a modern aesthetic interface and an AI-powered opponent, allowing players to compete against club members represented as bots. The project demonstrates the use of HTML, CSS, and JavaScript to build an interactive web game.

---

🚀 Features

- ✨ Clean and aesthetic glassmorphism UI
- 🧠 Smart AI opponent powered by the Minimax algorithm
- 👥 Compete against STEM Club members represented as bots
- 🎉 Confetti celebration when a player wins
- 📱 Mobile-friendly design
- 🎮 Local multiplayer mode (Play IRL)
- ⚡ Smooth animations and glowing winning cells

---

📱 How to Play

1. Enter your name when the game starts.
2. Choose a game mode:
   - Play IRL
   - Compete with Members
3. Take turns placing X and O on the board.
4. Get three in a row to win the game.
